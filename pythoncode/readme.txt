before you start (especially for the first time)
to make sure pypetter chromium download doesnt eror:
set PYPPETEER_CHROMIUM_REVISION=1299982 (use the latest revision, check at https://download-chromium.appspot.com/)
set PYPPETEER_DOWNLOAD_HOST=https://commondatastorage.googleapis.com

if dependencies error happen, do:
pip install requirements.txt
